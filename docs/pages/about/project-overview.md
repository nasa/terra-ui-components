# Project Overview

Terra UI Components is a collection of web components for working with Earthdata services.

## What is Terra UI?

Terra UI Components provides a comprehensive set of reusable web components that follow NASA's design standards and best practices. These components are framework-agnostic and can be used with any modern web framework or vanilla JavaScript.

## Key Features

- **Framework Agnostic**: Works with React, Vue, Angular, or vanilla JavaScript
- **Accessible**: Built with accessibility in mind, following WCAG guidelines
- **Customizable**: Fully customizable with CSS variables and themes
- **Dark Mode**: Includes built-in dark theme support
- **TypeScript**: Fully typed with TypeScript definitions
- **Open Source**: Available under an open source license
