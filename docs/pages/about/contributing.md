# Contributing

Thank you for your interest in contributing to Terra UI Components! We welcome contributions from the community.

## How to Contribute

### Reporting Issues

If you find a bug or have a feature request, please open an issue on our [GitHub repository](https://github.com/nasa/terra-ui-components).

### Submitting Pull Requests

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Submit a pull request with a clear description of your changes

### Code Style

Please follow the existing code style and conventions in the project. We use:

- ESLint for JavaScript/TypeScript linting
- Prettier for code formatting
- TypeScript for type safety

### Testing

Please ensure all tests pass before submitting a pull request. Include tests for new features when applicable.

## Questions?

If you have questions about contributing, please open an issue or reach out to the maintainers.
